# 地属性ドラフト カードプールデータ管理

## 1. プロジェクト概要

このリポジトリは、地属性ドラフトで使用するカードプール情報と枚数管理データを格納しています。

Excel（CSV）形式で作成されたデータを、プログラムでの利用やバージョン管理を容易にするため、YAML形式に変換して管理しています。

## 2. データバージョン

* **バージョン:** `7.34`
* **最終更新日:** 2025年12月11日（またはファイルをコミットした日付）

## 3. ファイル一覧

| ファイル名 | 概要 |
| :--- | :--- |
| `csv_to_yaml.py` | yamlファイル作成用のpythonファイル。
| `earth_draft_pool.csv` | カードプールデータのCSV。カード名、レアリティ、枚数、セクション（メイン/エクストラ）を格納。 |
| `earth_draft_pool.yaml` | メインとなるカードプールデータ。カード名、レアリティ、枚数、セクション（メイン/エクストラ）を格納。 |
| `README.md` | 本リポジトリの説明ファイル。 |

## 4. データ構造 (earth_draft_pool.yaml)

`earth_draft_pool.yaml` は、以下の構造でカード情報を管理しています。

```yaml
pool:
  name: "地属性ドラフト"
  version: 7.34
  cards:
    normal: # ノーマルカードのリスト
      - name: "TG ストライカー"
        count: 3
        section: "main"
      # ... 他のノーマルカード
    rare: # レアカードのリスト
      - name: "コスモス姫のご就寝"
        count: 1
        section: "main"
      # ... 他のレアカード
    initial_distribution: # 初期配布カードのリスト（大嵐、スクラップ・ドラゴン）
      - name: "大嵐"
        count: 6
        section: "main"
